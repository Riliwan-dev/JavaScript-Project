Modern Photo Gallery
=====================

What's included
- index.html — main page
- styles.css — styles (responsive, modern)
- script.js — gallery logic: lazy-load, lightbox, keyboard controls, search & filter, autoplay
- images/*.svg — 8 generated SVG placeholders (you can replace them with real photos)

How to use
1. Unzip and open index.html in a browser.
2. For real photos, replace the files in the images/ folder and adjust names in script.js or use filenames that match.

Features added
- Responsive masonry layout (CSS columns)
- Lazy-loading via IntersectionObserver
- Accessible lightbox with keyboard navigation
- Autoplay slideshow, shuffle, search, tag filter, download & fullscreen
- Smooth animations and modern UI

Customize
- Swap SVGs for JPEG/PNG and update the IMAGES array in script.js
- Add more tags and images quickly.
